### batch_calculate (on Linux)

1. Download https://github.com/kimwalisch/primesieve/releases/tag/v12.13
2. https://github.com/kimwalisch/primesieve/blob/master/doc/BUILD.md
3. Compile cpp program: ```g++ -O3 -march=native -std=c++17 -o prime_search prime_search.cpp -lprimesieve -lpthread```
4. Run ```python run.py```
